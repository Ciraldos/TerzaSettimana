EcmScript6 [2015]

Ecmascript (o ES) è lo standard per il linguaggio di scripting secondo le specifiche del documento ECMA-262 (vedi sito). Ecma è la realtà di riferimento per la stadardizzazzione del settore informatico. Ecmascript è uno standard pensato per garantire l'interoperabilità delle pagine sui browser.
Con l'avvento di EcmaScript6 sono state introdotte una serie di novità.

  • Già viste
  • let e const
  • Arrow function =>
  • nuovi metodi della libreria Math
  • Vedremo oggi
  • Operatore spread
  • Parametri rest
  • Destrutturare array|oggetti
  • Template literals
  • Nuovi metodi delle stringhe
  • Nuovi metodi array
  • Vedrete più avanti (7a settimana)
  • Class
  • Promises
